## Setup
Download [Node.js](https://nodejs.org/en/download/) and run this followed commands:

On front folder : 

``` bash
# Install dependencies (only the first time)
npm install
# Run the local front server
npm run dev
```

On back folder : 

``` bash
# Install dependencies (only the first time)
npm install
# Run the local back server
npm run dev
```
